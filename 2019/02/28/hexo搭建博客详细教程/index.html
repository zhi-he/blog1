hexo搭建博客详细教程

发表于 2019-02-28 | 分类于 教程

前言

一个偶然的机会，想自己搭建一个博客，在网上查了很多，最后决定用 hexo + github page，搭建后可以不到github上面直接编译成为静态文件，配置github page即可线上访问，无需购买域名，欢迎来访 一介码农。
我是一个前端菜鸟，感觉这个比较适合我，走了很多弯路，本着服务大众的态度，写下这个教程，第一次写博客，欢迎吐槽。

注：本教程只适用于windows系统

搭建前准备

  1. nodeJs
  2. git
  3. 编辑工具（VSCode 或 其他）

初始化基本博客

使用npm安装hexo脚手架，之前一直使用cnpm安装，后来发现cnpm有些意料之外的bug，决定使用npm，把源地址换为cnpm即可，在此推荐一个工具 nrm，nrm是npm源地址管理工具，直接npm install nrm -g安装，nrm use cnpm即可切换完成。
